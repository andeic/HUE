HUE API Documentation

.getGroup()

Returns the attributes of a specific group.

.getGroups()

Returns a list of all the groups of lights.

.getLight()

Returns the current attributes of a specific light.

.getLights()

Returns a list of all the lights on the network.

.groupOff()

Turns a group of lights off.

.groupOn()

Turns a group of lights on.

.init()

Initializes the library with the supplied values.

.lightOff()

Turns a specific light off.

.lightOn()

Turns a specific light on.

.swich()

Switches a specific light on or off depending on its current state.
