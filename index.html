HUE API Documentation

.init()

Initializes the library with the supplied values.

Light Functions

.breathe()

Performs one breathe cycle for a specific light.

.breathing()

Performs breathe cycles for a specific light for 30 seconds, or for a supplied amount of time less than 30 seconds.

.bri()

Changes the brightness of a specific light to a new brightness, within the valid range.

.color()

Changes the color of a specific light to a new color.

.colorLoop()

Activates or deactivates the color loop setting of a specfic light depending on its current state. Can also take an amount of time for the effect to automatically turn off.

.custom()

Sends a set of custom options to a specific light.

.getLight()

Returns the current attributes of a specific light.

.getLights()

Returns a list of all the lights on the network.

.off()

Turns a specific light off.

.on()

Turns a specific light on.

.swich()

Switches a specific light on or off depending on its current state.

Group Functions

.Gbreathe()

Performs one breathe cycle for a group.

.Gbreathing()

Performs breathe cycles for a group for 30 seconds, or for a supplied amount of time less than 30 seconds.

.Gbri()

Changes the brightness of a group to a new brightness, within the valid range.

.Gcolor()

Changes the color of a group to a new color.

.GcolorLoop()

Activates or deactivates the color loop setting of a group depending on the supplied value. Can also take an amount of time for the effect to automatically turn off.

.Gcustom()

Sends a set of custom options to a group.

.getGroup()

Returns the attributes of a group.

.getGroups()

Returns a list of all the groups of lights.

.Goff()

Turns a group of lights off.

.Gon()

Turns a group of lights on.

Color Conversion Functions

.rbgtoxy()

Converts a supplied rgb color value to an appropriate value in the CIE color space.

.rbgtohex()

Converts a supplied rgb color value to a hex triplet.

.hextorbg()

Converts a supplied hex triplet to a rgb color value.

.hextoxy()

Converts a supplied hex triplet to an appropriate value in the CIE color space.

Member Variables

.ajaxResult

Contains the returned json of the latest Philips Hue API call.

.groupurl

Contains the base url for the lights API of the Philips Hue bridge.

.lighturl

Contains the base url for the groups API of the Philips Hue bridge.
